# FilterBrowser
filterbrowser is a proxy from scramjet (the sucessor to uv) that is basically a browser but worse

# Main Features
- Open in Blob
- Open in about:blank
- History Cloak
- Exit tab Cloak
- Focus Proxy Cloak
- 20+ Unique Themes
- 750+ Games
- Powerful AI Assistant
- Panic Key
